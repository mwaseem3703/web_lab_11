# Nexus Digital - Professional Company Portfolio

A modern, high-performance company website built with **Laravel 11** and **Tailwind CSS**. This project features a clean, professional UI designed for software agencies, featuring interactive routes for Home, Profile, Clients, Products, and Contact Us pages.

## 🚀 Key Features

- **Modern UI/UX**: Built with a "Premium Agency" aesthetic using Tailwind CSS.
- **Blade Templating**: Optimized using a master Layout system for easy maintenance.
- **Responsive Design**: Fully compatible with Mobile, Tablet, and Desktop views.
- **Interactive Components**: Hover effects, glassmorphism navigation, and gradient headers.
- **Clean Routing**: Structured `web.php` for seamless page transitions.

---

## 🛠️ Tech Stack

- **Framework**: [Laravel](https://laravel.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (CDN-based for fast setup)
- **Icons**: Lucide React / SVG Icons
- **Fonts**: Inter Google Font

---

## 📂 Project Structure

- `resources/views/layout.blade.php` - The master template (Header/Footer).
- `resources/views/home.blade.php` - Interactive Hero section and stats.
- `resources/views/profile.blade.php` - Personal/Founder bio and skills grid.
- `resources/views/products.blade.php` - Service cards with hover animations.
- `resources/views/clients.blade.php` - Professional client logo grid.
- `resources/views/contact.blade.php` - Dual-column layout with working form UI.
